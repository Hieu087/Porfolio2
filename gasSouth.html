Gas South

Details of my works

Gas South app require a scanner for serial numbers with minimum response time so advanced technology are necessary. I single-handedly took the responsibility for this app.

View in store

1. Login Function

Login with 3 methods (4 for Iphone users):
- Login with Facebook: Using react-native-fbsdk module. Since Facebook has ended official support for React Native so this module is no longer usable.
- Login with Google: Using Google signin provider from Firebase.
- Login with Phone: Login with OTP confirmation by using provider from Google Firebase.
- Login with Apple ID (for Iphone user): Using react-native-apple-authentication module.

2. Serial Number Scanner

- Each gas cylinder will have their unique serial number, these serial numbers can be used to check for the origin of gas cylinders or create a record.
- To create a serial number scanner, I used react-native-camera module and Google Vision API to accomplish.
- After users have scanned a valid serial number, the system will navigate them to a new screen with the cylinder's info being displayed.
